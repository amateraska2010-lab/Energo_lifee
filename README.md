# AI Career Matchmaker 
Сервис для автоматического сопоставления студентов и вакансий.

###  Технологии:
- **Python & FastAPI**: Бэкенд и API.
- **OpenAI GPT-4o**: Анализ навыков и умный мэтчинг.
- **PostgreSQL**: Хранение профилей и истории развития.

###  Структура проекта:
- `/backend`: Логика обработки данных и AI.
- `/database`: Схемы таблиц базы данных.

###  Цель:
Помочь студентам закрыть "gap" (разрыв) между их навыками и требованиями реального IT-рынка.
